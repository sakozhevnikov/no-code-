# no-code-Приложение для анализа табличных данных с числовыми и/или категориальными переменными (Intelligent EDA System)

## Описание проекта

Система для разведочного анализа данных с возможностью:
- предобработки данных
- визуализации
- обучения моделей
- анализа признаков

## Стек
- Python
- streamlit
- Pandas
- numpy
- plotly
- openpyxl
- scipy
- Scikit-learn
- optuna

## Возможности
- загрузка и анализ табличных данных;
- автоматизированный разведочный анализ данных (EDA);
- визуализация распределений и зависимостей;
- обработка пропусков, выбросов и дубликатов;
- масштабирование и предобработка признаков;
- анализ дисбаланса классов и корреляций;
- обучение и сравнение ML-моделей;
- расчет метрик качества моделей;
- подбор гиперпараметров;
- многостраничный пользовательский интерфейс для работы с данными.
